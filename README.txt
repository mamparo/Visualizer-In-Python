Visualizer.py Version 0.7 04/30/2014

SUMMARY
-------
-Takes audio files and grabs the samples from the audio files

-Finds the highest and the lowest sample and creates sets of data

-Displays colors based on regions in the data

-Bar visualizer added to the code
